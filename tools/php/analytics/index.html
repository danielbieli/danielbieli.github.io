Installation.
Installation is super easy and takes just a few seconds.
Please just follow the steps below to install the application on any website.

1. Open "main.php" with a text editor and change the settings (e.g. Admin-Password) to your likings

2. Upload all files to a directory of your website (e.g. "../analytics")

That's it - there is nothing more to do. The installation is ready.
To add analytics to a page of your website, please follow the steps below:

1. Add this code to any page of your website you want analytics on:

    <script type="text/javascript" src="http://www.yourdomain.com/directory/analytics.js"></script>

    ·  Place the code right before the </body>-tag at the end of your page
    ·  Replace "yourdomain.com" with your own domain name
    ·  Replace "directory" with the directory name you uploaded the script to

2. Add the code to every page you want analytics on. That's it.
